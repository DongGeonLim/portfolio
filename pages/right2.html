Immersive Map Guide System

졸업 연구 프로토타입: 실시간 위치 데이터 기반 공간 조명 동기화

실시간 데이터 연동 시각 가이드 인터페이스

지정된 디지털 캠퍼스 맵 아키텍처 위에 하드웨어 인프라 기술을 시각적으로 결합한 가이드 시스템입니다. 좌측 하단의 디바이스 노즐에서 전송되는 연동 시퀀스를 기반으로, 콤팩트하게 압축된 원형(Circle) 스포트라이트 빔이 맵 내부의 특정 좌표나 이벤트 거점을 핀포인트로 조준하며 부드럽게 스캔합니다. 유저의 공간 대형 데이터와 기획 동선을 실시간 맵 레이어와 하드웨어 연출로 상호 결합하는 직관적인 실감형 인터랙션을 증명합니다.
